## 🚀 Featured Projects

### 🔬 CampusBot — Domain-Specific RAG Assistant
> Built a curriculum-aware RAG system trained on university course manuals — outperforming general-purpose LLMs on domain-specific queries by reasoning directly from source material.

- Designed a full **NLP ingestion pipeline**: PDF extraction → semantic chunking → dense embedding → vector database indexing across 100+ pages
- Integrated **Ollama + LLaMA** to serve grounded, hallucination-resistant answers — significantly reducing student manual search effort
- Architecture follows production RAG patterns: retrieval quality tuning, context window management, and response grounding

`Python` · `RAG` · `LangChain` · `LLaMA` · `Ollama` · `Vector DB` · `FastAPI`

---

### 📢 ADsPro — Automated Ad Management Platform
> Engineered a multi-role web platform with a structured 3-stage moderation workflow — automating the ad review lifecycle from submission to payment verification.

- Implemented **RBAC** across 3 user tiers (Advertiser / Moderator / Admin) with fully isolated dashboards
- Built a secure transaction ID verification flow that substantially reduced unverified ad placements
- Replaced manual moderation bottlenecks with an automated role-gating pipeline

`Full-Stack` · `RBAC` · `Workflow Automation` · `REST API` · `Firebase`

---

### 📱 Flutter Application Suite
> Shipped a suite of cross-platform mobile applications demonstrating end-to-end product delivery — from UI design to Firebase backend integration and state management.

- **Task Manager** — real-time CRUD with Firebase sync and offline support
- **POS Billing System** — streamlined checkout flow with receipt generation
- **BMI Tracker · Calculator · Portfolio App** — responsive UI across Android and iOS

`Flutter` · `Dart` · `Firebase` · `State Management`
